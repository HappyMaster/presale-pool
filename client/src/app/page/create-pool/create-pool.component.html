Create An Investor Pool

1 Define

EtherPool has no cap on how much ether you can raise or how many investors can invest. However, there are relatively high gas costs to create Whitelists. The network also charges relatively high gas costs to edit the max/min values after the pool is launched.

All fields are optional.

2 Launch

Copy the data below into MyEtherWallet in a new browser. MyEtherWallet will generate a Transaction ID for you to input in the bottommost input field on this form. Click submit and your pool will launch!
